# Mellowfmt

An interactive playground for exploring and comparing code formatting options between [Prettier](https://prettier.io/) and [Oxfmt](https://oxc.rs/docs/guide/usage/formatter/config-file-reference.html).

## Overview

Mellowfmt allows you to:
- Browse Prettier and Oxfmt formatter options with live code previews
- Compare options side-by-side and understand how each setting affects output
- Export your configuration for use in your projects

## Stack

- **Framework**: React 19 with [TanStack Start](https://tanstack.com/start)
- **Routing**: [TanStack Router](https://tanstack.com/router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components**: [Shadcn/ui](https://ui.shadcn.com/) (built on Radix UI)
- **Build Tool**: [Vite](https://vite.dev/)
- **Language**: TypeScript

## Getting Started

```bash
bun install
bun dev
```

## Building for Production
